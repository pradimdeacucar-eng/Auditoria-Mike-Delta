### **FASE B: 300 NOVAS UNIDADES (Anos 2-3)**

```

Year 3 (2029): 50 unidades (ajustes)

└─ Sem morada: Prioridade máxima

```

**Fase 2.1: Preparação (Mês 1-2)**
```
ATIVIDADE: Preparar lançamento do PMCMV
RESPONSÁVEL: Secretária de Habitação + CEF

1. Contatar CEF (Caixa Econômica) Salvador
   └─ Apresentar interesse + demanda
   
2. Identificar terrenos municipais
   ├─ Prado possui terras? Sim (verificar)
   ├─ Necessário: 60-70 ha (3 lotes de 20ha c/)
   └─ Localização: Próximo infraestrutura
   
3. Elaborar Projeto Urbanístico
   ├─ 100 unidades = 30 ha (com ruas, praças)
   ├─ Projeto técnico completo
   ├─ Levantamento planimétrico
   └─ Custos: R$ 30 mil
   
4. Identificar demanda
   ├─ Publicar edital de interesse
   ├─ Reunir 1.000+ famílias interessadas
   ├─ Seleção prévia (maior carência)
   └─ Listagem de 500 pré-selecionadas

CUSTO FASE 2.1: R$ 50 mil
```

**Fase 2.2: Aprovações e Financiamento (Mês 3-4)**
```
ATIVIDADE: Aprovação federal + contrato de financiamento
RESPONSÁVEL: CEF + Município

PROCEDIMENTO:
1. Submeter projeto à CEF
   ├─ Documentação técnica
   ├─ Planilha orçamentária
   ├─ Capacidade municipal (contrapartida)
   └─ Prazo: 30 dias análise
   
2. CEF analisa e aprova
   ├─ Se aprovado: Empenho de recursos
   ├─ Liberação: 40% adiantado, 60% por obra
   └─ Resultado: Contrato assinado
   
3. Município formaliza contrapartida
   ├─ 10% do investimento (R$ 2 milhões)
   ├─ Terreno cedido (valor = contrapartida)
   ├─ Custos de aprovação municipal
   └─ Documentação dos 300 beneficiários
```
