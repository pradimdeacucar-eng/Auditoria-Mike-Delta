```
BAIXO (FINANCIAMENTO SEGURO)

RAZÕES:
✓ 70% garantido (federal = caixa que executa)
✓ Repasse automático (orçamento)
✓ Contrapartida viável (arrecadação cresce)
✓ Possibilidade renegociação (prazos flexíveis)
✓ Resultado gera receita (IPTU+ICMS compensam)

├─ Se cair arrecadação -20%: Ainda consegue 70% contrapartida
├─ Se atrasarem 1 ano: Desliza cronograma (não cancela)
├─ Se economia desacelerar: Receita ainda cresce (habitação = garantia)
└─ Mínimo viável: 30% do cronograma (300 casas mínimo)

```

✓ REURB já executado em 500+ municípios BR

✓ Metodologia testada (Bahia executa há 15 anos)

├─ Conflito fundiário → MP + cartório resolvem
