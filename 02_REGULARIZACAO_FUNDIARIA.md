## ⚙️ COMO FUNCIONA A REURB

### **3 CAMINHOS POSSÍVEIS**

```
┌─────────────────────────────────────────────────────────────┐
│                      COMO REGULARIZAR                       │
└─────────────────────────────────────────────────────────────┘

1. REURB-S (Áreas Públicas)
   ├─ Quem regulariza: Município
   ├─ Custo: NENHUM (gratuito)
   ├─ Documentos: Menos
   ├─ Prazo: 6-12 meses
   └─ Melhor para: Maioria dos casos

2. REURB-E (Áreas Privadas - Boa Fé)
   ├─ Quem regulariza: Proprietário OU Município
   ├─ Custo: Proprietário paga cartorial
   ├─ Documentos: Mais
   ├─ Prazo: 12-18 meses
   └─ Melhor para: Ocupações com proprietário cooperativo

3. REURB COLETIVA (Vários proprietários)
   ├─ Quem regulariza: Administrador (Município)
   ├─ Custo: Rateia entre famílias
   ├─ Documentos: Complexo
   ├─ Prazo: 18-24 meses
   └─ Melhor para: Grandes ocupações mistas
```

## 🎯 PASSO-A-PASSO: REURB-S (RECOMENDADO PARA PRADO)

### **ETAPA 1: IDENTIFICAÇÃO E LEVANTAMENTO (30 dias)**

#### **Passo 1.1: Constituir Comissão de Regularização**

**Composição:**
```
Presidente: Secretária de Habitação
Membros:
├─ Secretária de Planejamento (zoneamento)
├─ Secretária de Finanças (dados cadastrais)
├─ Controlador Interno (transparência)
├─ Assessor Jurídico (questões legais)
├─ Engenheiro (levantamento técnico)
└─ Assistente Social (comunidade)

Responsabilidades:
✓ Coordenar todas as fases
✓ Aprovações e assinaturas
✓ Comunicação com comunidade
✓ Resolução de conflitos
```

**Tempo:** 1 semana  
**Custo:** NENHUM (usa estrutura municipal)

---

#### **Passo 1.2: Mapeamento Técnico**

**Material necessário:**
```
Equipamento:
├─ GPS (smartphone ou aparelho profissional)
├─ Drone (opcional, facilita visualização)
├─ Câmera fotográfica
├─ Prancheta e material de anotação
└─ Planilha de cadastro (Excel)

Procedimento:
1. Marcar limites da área (4 pontos de GPS)
2. Fotografar panorâmica (4 ângulos)
3. Marcar cada lote (ponto central + 2 cantos)
4. Registrar ocupação atual (construído/vago)
5. Notar características (água, luz, acesso)
6. Coletar coordenadas UTM (para cartório)

Resultado:
✓ Planta com todos os lotes
✓ Coordenadas de cada um
✓ Documentação fotográfica
✓ Descrição técnica completa
```

**Tempo:** 2 semanas (equipe de 2-3 pessoas)  
**Custo:** R$ 2-5 mil (material + diárias)

---

#### **Passo 1.3: Cadastro de Famílias**

**Informações por família:**
```
DADOS PESSOAIS:
├─ Nome completo (e cônjuge)
├─ CPF e RG
├─ Data de nascimento
├─ Estado civil
├─ Profissão
└─ Telefone/email

DADOS DO LOTE:
├─ Localização (rua, número)
├─ Coordenada GPS
├─ Área do lote
├─ Metragem construída
├─ Idade da ocupação
└─ Benfeitorias (água, luz, etc)

DOCUMENTAÇÃO:
├─ Comprovante de renda
├─ Comprovante de residência
├─ Histórico de ocupação
└─ (se houver) Documentos anteriores

CLASSIFICAÇÃO:
├─ REURB-S ou REURB-E?
├─ Há conflito?
├─ Qual a situação?
└─ Necessita reassentamento?
```

**Processo:**
```
1. Equipe vai de porta em porta
2. Aplica formulário de cadastro
3. Fotografa beneficiário + família
4. Coleta cópias de documentos
5. Classifica ocupação
6. Registra banco de dados municipal

Importante:
✓ Criar clima de confiança
✓ Explicar benefício (titulação)
✓ Ouvir reclamações
✓ Anotar obs. especiais (saúde, educação)
```

**Tempo:** 3 semanas (equipe de 2-3 assistentes sociais)  
**Custo:** R$ 5-10 mil (diárias + material)

#### **Passo 3.2: Levantamento Planimétrico (Cartório)**

**O que é:**
```
Levantamento técnico feito por topógrafo/cartório
para determinar exatamente onde está cada lote,
com coordenadas UTM e descrição legal.
```

**Processo:**
```
1. Topógrafo vai ao local com aparelhos
2. Marca cada lote com precisão (±1 metro)
3. Coleta coordenadas GPS (latitude/longitude)
4. Determina limites físicos
5. Elabora planta com descrição de cada lote

Resultado:
├─ Planta de divisão (scale 1:500)
├─ Descrição de cada lote
├─ Coordenadas UTM
├─ Área de cada um (m²)
└─ Localização exata para cartório
```

**Tempo:** 3-4 semanas  
**Custo:** R$ 15-25 mil (topógrafo)

### **ETAPA 5: ACESSO A SERVIÇOS (30 dias)**

#### **Passo 5.1: Ligação de Água**

**Processo:**
```
1. Beneficiário vai até SAAE (Serviço Autônomo Água Esgoto)
2. Apresenta documento de legitimação
3. Preenche requerimento
4. Paga taxa de ligação (R$ 100-300)
5. Técnico instala hidrômetro
6. Ativa ligação

Resultado:
├─ Água encanada em casa
├─ Medidor registrado
├─ Conta em seu nome
└─ Acesso a crédito (bem de raiz)
```

**Tempo:** 15 dias  
**Custo:** R$ 200 por família

---

#### **Passo 5.2: Ligação de Energia**

**Processo:**
```
1. Beneficiário vai até COELBA (Companhia Elétrica)
2. Apresenta documento de legitimação
3. Preenche requerimento
4. Paga taxa de ligação (R$ 150-400)
5. Técnico instala hidrômetro/medidor
6. Ativa ligação (24h)

Resultado:
├─ Energia elétrica legal
├─ Medidor em nome dele
├─ Conta registrada
└─ Acesso a crédito
```

**Tempo:** 10 dias  
**Custo:** R$ 250 por família

---

#### **Passo 5.3: IPTU (Imposto Predial)**

**Processo:**
```
1. Prefeitura registra no cadastro municipal
2. Beneficiário pode optar:
   
   OPÇÃO A: Pagar IPTU
   ├─ Valor: ~R$ 80-150/ano (lote urbano)
   ├─ Benefício: Voto para melhorias
   └─ Prioridade: Ruas, escolas

   OPÇÃO B: Isenção por renda (se <5 SM)
   ├─ Processo: Requerimento + comprovação renda
   ├─ Duração: 3-5 anos (renovável)
   └─ Após: Retorna IPTU normal

Resultado:
├─ Registro no cadastro imobiliário
├─ Estatísticas de urbanização
└─ Receita para município
```
